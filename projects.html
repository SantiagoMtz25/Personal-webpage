IoT, Sensor connected to cloud

In this project I created a project using a proximity sensor connected to cloud, the sensor connected to the cloud using a firebase, sensor could be activated and deactivated from distance through a webpage and assistant Alexa. Information related to sensor's activity was stored in a SQL database. Circuit was constructed and programmed with NodeMCU and Arduino.

It represented a big challenge sinca I had never worked before with IoT project but I was able to understand how many things can be done when combining programming with hardware.

Webpage connected to MySQL database

Helped building an interactive web page for my university's student annual survey seeking to increase student participation. I used PHP to create and destroy user sessions when an account was logged in and logged out, to check the credentials and validate the type of users, the PHP code executed a query in a MySQL Database and searched the data of email and password given by the user to see if it exists.
When granting access to an user depending on the type of their account they would be logged to a specific page. Besides, the PHP code, the pages were built using pure HTML and CSS.
